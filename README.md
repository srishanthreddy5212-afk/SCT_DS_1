# SkillCraft---Internship

## 📌 Overview

This repository contains the projects completed during my **SkillCraft Technology Data Science Internship**. The internship focused on developing practical skills in **data visualization, exploratory data analysis (EDA), machine learning, and accident data analysis** using Python and popular data science libraries.

Each task demonstrates the application of data science techniques to solve real-world problems and gain meaningful insights from data.

---

## 📂 Repository Contents

### 📊 Task 1 – Population Data Visualization

- Created bar charts and histograms to visualize population distribution.
- Analyzed country-wise population using the World Bank Population Dataset.
- Explored trends in population across different countries.

**Tools Used:** Python, Pandas, Matplotlib

---

### 🧹 Task 2 – Titanic Dataset (Data Cleaning & EDA)

- Performed data cleaning by handling missing values.
- Conducted Exploratory Data Analysis (EDA).
- Visualized relationships between variables such as age, gender, passenger class, and survival.
- Identified key factors affecting passenger survival.

**Tools Used:** Python, Pandas, Matplotlib, Seaborn

---

### 🌳 Task 3 – Decision Tree Classification

- Built a Decision Tree Classifier using the Bank Marketing Dataset.
- Preprocessed categorical data using Label Encoding.
- Trained and evaluated the classification model.
- Visualized the Decision Tree and measured model accuracy.

**Tools Used:** Python, Pandas, Scikit-learn, Matplotlib

---

### 🚗 Task 4 – Traffic Accident Data Analysis

- Analyzed traffic accident data to identify accident patterns.
- Explored the effects of weather conditions, road conditions, and time of day on accidents.
- Visualized accident hotspots and contributing factors.
- Generated insights through exploratory data analysis and visualizations.

**Tools Used:** Python, Pandas, Matplotlib, Seaborn
